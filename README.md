
## Description

Basic [NestJs](https://github.com/nestjs/nest) CRUD APP

## Installation

```bash
$ npm install
```
## Database Configuration

```bash
# To run Database. 
$ docker-compose up -d
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
